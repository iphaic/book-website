# book-website
a website built with php &amp; SQL running backend and HTML with css for the frontend. For a school project

To use:
1. Download XAMPP and install it
2. Open `http://localhost/phpmyadmin` in your browser, click the SQL tab at the top, and copy/paste the `db.sql` into there and press Go
3. Open the XAMPP control panel and start Apache and MySQL
4. Open `http://localhost/online_bookstore/index.php` in your browser
5. You should be good!
