# book-website
a website built with php &amp; SQL running backend and HTML with css for the frontend. For a school project

To use:
1. Download XAMPP and install it
2. Place this repository in `C:\xampp\htdocs\online_bookstore`. Your path to `index.php` should look like: `C:\xampp\htdocs\online_bookstore\index.php`
3. Open the XAMPP control panel and start Apache and MySQL
4. Open `http://localhost/phpmyadmin` in your browser, click the SQL tab at the top, and copy/paste the `db.sql` into there and press Go
5. Open `http://localhost/online_bookstore/index.php` in your browser
6. You should be good!
